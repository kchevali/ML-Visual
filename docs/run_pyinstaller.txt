In case the spec file gets deleted add these data files to the spec file:

project_files=[
    ( 'examples/*', '.' ),
    ( 'assets/*', '.' )
]

app = BUNDLE(exe,
         name='TeachApp.app',
         icon=None,
         bundle_identifier="com.chevalier.teachapp.zip")

Next add project_files to the data args in Analysis (~line 14)
